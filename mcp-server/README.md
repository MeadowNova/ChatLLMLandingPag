# Copilot MCP Server

This is a Model Context Protocol (MCP) server for workspace tool access, built in TypeScript.

## Usage

1. Build the server:
   ```bash
   cd mcp-server
   npx tsc
   ```
2. Run the server:
   ```bash
   node build/index.js
   ```

## SDK Reference
See the [Model Context Protocol SDK](https://github.com/modelcontextprotocol/create-python-server) for more details.
